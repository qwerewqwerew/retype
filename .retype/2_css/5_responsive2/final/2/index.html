display flex order 속성 정리 order : 아이템들의 순서를 변경할 때 사용 order 속성은 정수 값을 사용하며, 디폴트 값은 0. order 속성값이 높을수록 해당 아이템은 이전 아이템보다 뒤에 위치
break-point device 제목3 제목4
360~767 Mobile --- ---
768~959 Tablet --- ---
960~ Desktop --- ---
